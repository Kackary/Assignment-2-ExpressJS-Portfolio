# Personal Portfolio Website (2)

A personal portfolio website built with Express.js and Node.js for INFR3120 Assignment 2.

## Pages
- Home – Welcome page  
- About Me – Information about me  
- Projects – My projects  
- Contact Me – Contact form and contact information  

## Features
- Purple gradient navigation bar  
- Responsive design using Bootstrap
- Easy navigation through links  
- Functional contact form  
- GitHub and LinkedIn links in the footer  

## Technologies Used
- Express.js
- Node.js
- EJS
- Bootstrap
- Font Awesome
